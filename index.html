Overview

A play application that provides an encylopedia of surfers.

Installation

Download the zip file, unzip, cd into the directory, invoke the Play console, and execute the run command to see the application at http://localhost:9000

Usage

Use the navbar to find interior pages with with a form to add a new contact.
